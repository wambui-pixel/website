Abstract Machines Products for the Cloud–Edge Continuum

Abstract Machines develops open-source infrastructure for IoT and AI — enabling device-to-cloud connectivity, secure workload orchestration, and real-time data processing. From microcontrollers to the cloud, our tools—Magistrala, SuperMQ, Propeller, and Rex—power the entire spectrum of connected intelligence.

Software

Magistrala – Complete IoT Platform

Magistrala offers full-stack IoT capabilities—from device identity to data integration. With fine-grained access control, multi-tenant domains, and secure APIs, it’s your go-to platform for managing connected infrastructure at scale.

Go to Magistrala

SuperMQ – Lightweight IoT Messaging

SuperMQ is a low-footprint, high-performance messaging engine optimized for IoT. It supports MQTT, CoAP, WebSocket, and more—ensuring fast, reliable communication from devices to cloud or edge systems.

Go to SuperMQ

Propeller – Serverless Orchestration

Propeller enables secure orchestration of WebAssembly functions across edge and cloud. Deploy serverless workloads on demand, with remote attestation, lightweight lifecycle control, and runtime portability from cloud VMs to microcontrollers. Ideal for modular, policy-driven AI and compute automation.

Go to Propeller

Hardware

S0 – Low-power RISC-V IoT Gateway

S0 is a low-power IoT gateway based on RISC-V architecture, designed for constrained environments. It provides efficient connectivity and processing for IoT devices with minimal energy consumption, making it ideal for edge deployments.

Go to S0 Gateway

S1 – RISC-V Linux Gateway

S1 is a powerful Linux-based IoT gateway built on RISC-V architecture. It offers robust connectivity, advanced processing capabilities, and scalability for industrial IoT applications, enabling seamless integration with cloud and edge systems.

Go to S1 Gateway
